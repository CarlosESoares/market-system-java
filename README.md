# 🛒 Market System – Sistema de Gestão Comercial

Sistema desenvolvido em Java como projeto integrador do curso técnico em informática, com foco na gestão de produtos, vendas e estoque.

## 🚀 Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Integração com banco de dados
- Operações CRUD completas

## 🛠️ Tecnologias utilizadas

- Java
- JDBC (ou o que você usou)
- Banco de dados relacional (MySQL/SQL Server/etc)
- Git/GitHub

## 📂 Estrutura do projeto

- Model → Representação dos dados
- DAO → Acesso ao banco de dados
- Service → Regras de negócio
- View → Interface do usuário (se tiver)
